Mini-Project

Radio Controlled RGB Lighting

Introduction

You will be programming an Arduino called the "Nano". This is an Arduino board with the same microcontroller chip as the Uno you have been using, but on a smaller board. This makes it easier to fit into small projects.

You'll use three potentiometers to control the R, G and B levels of a radio-controlled light. Your Arduino will read the light levels, then send them over a radio link to another Arduino controlling the light.

Basics

Here are the programming tasks that you'll be doing:

  1. Writing the code to read the R, G, and B controls
  2. Downloading and installing the radio software library
  3. Writing the code to send the R, G and B levels over the radio link

Setting up the sketch

  • Start a new Arduino sketch
  • At the top of the sketch, use #define to create three pin numbers. RED_PIN should be A0, BLUE_PIN should be A1 and GREEN_PIN should be A2.
  • In the setup function, use pinMode to make these pins OUTPUTs.

Reading the controls

  • In the loop, create three new variables: red, green and blue.
  • They should have the int datatype
  • Use analogRead to set red to the value of the RED_PIN pin.
  • Do the same for green and blue.

Testing the code

Let's test what you've done so far and make sure we're reading the values correctly.

  • Add the Serial.begin(9600) line to your setup function.
  • Use Serial.print and Serial.println to send the red, green and blue values back to the serial monitor
  • Turn each of the potentiometers. The numbers for each colour should decrease to 0 as you turn them anti-clockwise, and increase to 1023 as you turn them clockwise.

    • Radio Software

    There is a small, cheap radio transmitter inside your box, and a matching receiver inside the RGB light.

    The Arduino IDE doesn't come with software for this, so you'll need to install it.

    Download this ZIP file.

    In the Arduino IDE, go to Sketch... Include Library... Add .ZIP Library. Select the ZIP file you just downloaded. You can now use the radio software in your sketch.
