Mini-Project

Morse Code Beeper

Introduction

You've decided to learn Morse Code! To help you, you're going to build a device that tests your morse code knowledge.

The device will play dots-and-dashes at you and you'll then guess the letter by typing it in over the serial port.

Basics

It's always helpful to break a project down into small tasks. Here are the programming tasks that you'll be doing:

  1. Writing to code to play the right Morse code tones for a letter through a speaker
  2. Writing the code to pick a random letter from A to Z and play it
  3. Asking the user for their guess at which letter it was
  4. Telling the user if they got it right or not

Setting up the sketch

To help you, we've written a file that has all the letter A-Z in morse code.

  • First, start a new sketch and save it as morse_code_beeper
  • Download this file and save it into your sketch folder.
  • Restart the Arduino IDE and open the sketch again. You should see the downloaded file in another tab inside the IDE window
  • Click the tab to read the file. Each letter has a list of variables (called an array) that holds DOT or DASH.
  • We put a zero at the end of the array to mean "no more sounds"

Playing Tones

You'll need to write a new function to play the tone for a letter.
